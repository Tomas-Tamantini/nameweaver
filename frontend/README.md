# Frontend

## Setup

From this folder:

```bash
npm install
```

## Commands

- Start the local dev server:

  ```bash
  npm run dev
  ```

- Build for production:

  ```bash
  npm run build
  ```

- Preview the production build locally:

  ```bash
  npm run preview
  ```

- Check linting, formatting and tests:
  ```bash
  npm run check
  ```
